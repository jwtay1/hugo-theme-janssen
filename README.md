# Janssen (A Hugo theme)

Janssen is a theme for [Hugo](https://gohugo.io). The goal of this theme is to generate a documentation-like website for the [Bioimaging Wiki](https://bioimaging.wiki).

## Features

* Minimalist
* Typography
* Beautiful code blocks
